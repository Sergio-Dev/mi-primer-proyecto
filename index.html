Introduccion a Git

Crear mi primer repositorio con Git

Pasos a seguir:

  1. Instalar git
  2. Primera configuracion
    • git config --global user.name "Nombre Apellido"
    • git config --global user.email "email@mail.com"
  3. Crear una carpeta para guardar los repositorios
  4. Crear una carpeta para guardar un repositorio
  5. Crear varios archivos en la ultima carpeta creada
  6. Inicializar el repositorio
    • git init
  7. Agregar los archivos creados a la Staging Area (Zona Intermedia)
    • git add "nombre-del-archivo"
  8. Comprobar que los archivos creados estan en la Staging Area (Zona Intermedia)
    • git status
  9. Enviar los archivos al repositorio
    • git commit -m "mensaje del commit"
